# 学生管理系统

***

###功能介绍
***

* 实现对学生信息的录入
* 查看所有学生的信息
* 精确查找学生
* 修改学生信息
* 删除学生信息

        录入学生的学号,姓名,性别,年龄,专业.并保存在E:\students.txt当中.保存方式为二进制.所以无法直接查看学生的
        信息.要使用管理系统才能查看.
